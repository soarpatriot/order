An base front-end project for the stylus, jade gulp 
## Prerequisites

* NodeJS / NPM
* Gulp

```
npm install -g gulp
```

## Development instructions

### Install Dependencies

```
npm install
bower install
```

### Run Dev Environment

```
gulp
```

### Turn off first run rebuilds and endless lifereload updates on jade change:

```
gulp --light
```

<!-- ### Test

```
gulp test
``` -->

### Prepare Assets for Deployment

```
gulp deploy
```
